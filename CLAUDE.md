## Project Overview

Personal website and blog for Alejandro U. Alvarez, built with Hugo static site generator and styled with Tailwind CSS. The site features a bold, magazine-style design with lots of negative space, clean typography, and accent lines for visual interest.

**Live site**: https://aurbano.eu
**Repository**: https://github.com/aurbano/aurbano.eu

## Development Commands

### Initial Setup
```bash
# Clone repository
git clone https://github.com/aurbano/aurbano.eu.git

# Install dependencies
npm install

# Install Hugo (macOS)
brew install hugo
```

### Local Development
```bash
# Run local development server
hugo serve

# Build static site (outputs to public/)
hugo build
```

**Important**: When making changes to templates that use new Tailwind utility classes, always rebuild to regenerate the CSS. Hugo's PostCSS pipeline processes Tailwind on build, so new classes won't appear until you rebuild.

### Content Management
```bash
# Create a new post (uses archetype from archetypes/post.md)
hugo new post/YYYY-MM-DD-post-title.md

# Create a new project
hugo new projects/project-name.md
```

### Thumbnail Color Extraction
When adding new post thumbnails, run the color extraction script to update the dominant colors used for homepage project tiles:

```bash
# From project root
python3 scripts/colors.py
```

**Requirements**: `pip install colorthief`

**How it works**: Extracts dominant RGB color from each image in `assets/thumbnails/` and saves results to `data/colors.json`.

## Architecture

### Design System
The site uses a **magazine-style design** with:
- **Bold colors**: Accent pink (#FF3366), secondary purple (#6366F1), yellow (#FFD600), teal (#00D9C0)
- **High contrast**: Black text (#0A0A0A) on off-white background (#FAFAFA)
- **Large typography**: Headings use huge sizes (7xl-9xl) with tight tracking
- **Negative space**: Generous padding and margins throughout
- **Decorative lines**: Horizontal and vertical accent lines for visual interest
- **Tailwind CSS**: All styling uses Tailwind utility classes with custom components

### Content Structure
- `content/post/`: Blog posts (markdown files with front matter)
- `content/projects/`: Project pages
- `content/about.md`: About page
- `content/links/`: Link pages

### Front Matter for Posts
Posts require the following front matter:
```yaml
title: "Post Title"
description: "Brief description"
date: "YYYY-MM-DD"
thumbnail: filename.png  # Must exist in assets/thumbnails/ for homepage display
categories:
- Category Name
tags:
- Tag1
- Tag2
outdated: true  # Optional: shows warning banner on old posts
```

### Layout System
- `layouts/index.html`: Magazine-style homepage with hero section and project grid
- `layouts/_default/`: Default templates (single.html, list.html, etc.)
- `layouts/post/single.html`: Blog post template with breadcrumbs and related posts
- `layouts/partials/`: Reusable components (head.html, footer.html, list-item.html, etc.)
- `layouts/shortcodes/`: Custom Hugo shortcodes

### Available Shortcodes
- `{{< resource "path/to/file" >}}`: Include resources from assets
- `{{< resourceFigure "path" >}}`: Resource with figure wrapper
- `{{< youtube "video-id" >}}`: Embed YouTube videos
- `{{< youtubeFigure "video-id" >}}`: YouTube with figure wrapper
- `{{< codepen "pen-id" >}}`: Embed CodePen
- `{{< math >}}` and `{{< math-inline >}}`: Math rendering

### Styling
- `assets/css/main.css`: Main Tailwind CSS file with custom components and utilities
- `tailwind.config.js`: Tailwind configuration with custom colors and spacing
- `postcss.config.js`: PostCSS configuration for Tailwind processing

### Key CSS Classes
Magazine design components:
- `.magazine-hero`: Full-screen hero section
- `.magazine-section`: Standard content section with padding
- `.project-tile`: Featured project cards with hover effects
- `.accent-line`: Colored horizontal divider lines
- `.grid-line-vertical`/`.grid-line-horizontal`: Decorative grid overlays
- `.tag`: Pill-shaped tag badges
- `.nav-link`: Navigation links with underline animation

### Assets
- `assets/css/main.css`: Tailwind CSS with custom components
- `assets/thumbnails/`: Post thumbnail images
- `assets/posts/`: Post-specific assets

### Data Files
- `data/colors.json`: Dominant colors for thumbnails (generated by scripts/colors.py)

### Static Files
Files in `static/` are copied directly to the output without processing.

## Configuration

Hugo configuration is in `config.yaml`:
- Site metadata (title, author, description)
- Menu configuration (main menu and social icon links)
- Syntax highlighting settings
- Taxonomy configuration (posts, categories, tags)

## Build Output

The `hugo build` command generates the static site in `public/` directory. This directory is deployed to production.

## Design Philosophy

The magazine aesthetic emphasizes:
1. **Visual hierarchy**: Large headings, clear sections
2. **Whitespace**: Breathing room between elements
3. **Bold accents**: Strategic use of bright colors
4. **Typography**: Strong, readable fonts with varied sizes
5. **Minimalism**: Clean layouts without clutter
