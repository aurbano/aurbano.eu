How does it work?

Well the idea is relatively simple: Add CSS-hidden fields that a user won't see, but a spam-bot will fill. Then using php check if they were filled, and if they were: block the spammer! The CSS would be something as simple as: 
.noshow { display:none; }
The HTML form should include something like this:
And finally the php would be simply: 
if ($_POST['leaveblank'] == '' && $_POST['dontchange'] == 'http://') {
 // accept form submission
}
The only possible drawback would be that some users with CSS disabled would see the "hidden" form fields, and thus would be quite puzzled. The good thing is that the percentage of users with CSS disabled is so low that I wouldn't even care about that issue... I like this idea so much that I'm writing a WP plugin to do this for sure.
